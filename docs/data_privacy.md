# 数据与隐私说明

## 公开数据

公开仓库中的 `data/metadata_sample.csv` 是脱敏样例数据。它保留了 79 份问卷中用于复现统计图表的选择题字段，但移除了：

- IP、UA、Referrer 和地理位置；
- 开始时间、结束时间和答题时长；
- 开放题原文；
- 清洗标记及其他非必要元数据。

“其他用途”字段只保留是否填写的信息，原始自由文本统一替换为脱敏标记，以复现选择次数而不公开原文。

样例数据仅用于演示系统的数据分析能力，不应被视为具有代表性的教育研究数据集。

## 本地运行数据

用户在系统内提交的反馈会写入 `data/feedback.csv`。该文件已加入 `.gitignore`，首次提交反馈时由系统自动创建，不会被默认提交到 Git。

## 上传数据

调研数据页支持用户上传 CSV。使用者应避免上传不必要的个人信息；若数据包含个人信息，应在分析前获得授权并完成脱敏。

## 研究与伦理边界

EduAI-Guard 是教学与自查工具，不是正式的学术诚信判定系统。公开样例和系统评分不能替代学校政策、伦理审查或专业研究方法。
